Here from now you can learn more about Firefox OS Mobile application development. This series of tutorials will help you to learn the basics of Firefox OS application development.  It is one of the emerging Mobile OS Platform in the growing countries. There are lot of fans for this due to the simple User Interface and less priced mobile phone. You can know more about this in their Official Site

For developing the applications for the Firefox OS we need not have much knowledge. It is enough  to have some knowledge about HTML, CSS and Javascript. If you dont have hands-on this technologies no need to worry. Even without Programming you can develop mobile applications.

Yes you can. With the help of Mozilla Appmaker
This tool is developed by Mozilla Labs. It is one of the easiest development tool we can find. You can develop the applications and at the same time you can publish it without adding anything.
If you know some about web technologies. It will be great you can write your own code and start developing mobile applications. Below are some of the check List you need.
  1. Firefox Browser. Currently Firefox 29 is latest.
  2. Firefox OS simulator
  3. Notepad++ or any text editor
That's it you can now start our development. Firefox OS simulator needs to be installed from Firefox Browser. Make it correctly.
In the upcoming post you can know how to develop simple HTML page. And see it on simulator.
