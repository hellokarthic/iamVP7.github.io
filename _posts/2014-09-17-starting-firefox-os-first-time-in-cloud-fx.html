In my last post I have shared about Unboxing Cloud FX. Now I would like to share the basic setup of the device I did for the first time when I launched my phone.

Cloud FX showing Firefox OS logo on Start up

 By default the language will be selected as English. We can change to our desired language available for that particular device. We can either choose Hindi or Tamil in India as of now.
Default lang selection

Selecting Tamil as the default language

 After we start the device for the first time, we are in position to select the correct time and correct date of the device you should note that it is available in Tamil so it is easy for any one with Tamil knowledge to use the device.
Selecting date in Firefox OS
Selecting Time in Firefox OS

There are some string like AM and PM which should be localised. If you find anything like this please visit Bugzilla to tell your problem. Lot of volunteers will help you to solve this problem.
