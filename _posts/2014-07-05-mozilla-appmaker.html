Mozilla Appmaker is a very simple tool for those who are not aware of coding but would like to develop HTML5 based mobile applications. It is a free and open source tool which lets you to drag and drop individual components for our application. We can also contribute to their codebase of the Appmaker. Their official repo is at Github. Check it

In simple Appmaker is browser based tool used for creating simple mobile applications for ourselves for free. You easily drag and drop lot of components which you need for developing your personal application, as of now you cannot expect to develop apps like Angry birds.

Developing is very easy now
We already know it is possible to do drag and drop but to make it still easier just search some others apps and do little modification and give the name which you feel and share it, it will be become yours. Yes we can easily do

My First application for you all readers.

Would like to contribute
There are different number of ways one can contribute to Appmaker.
  • Easiest way is sharing a word about this to friends. So they can try and learn.
  • Join the mailing list
  • Host a party and teach others.
  • When developing if something does not work, file a bug here
Appmaker is created with a beautiful aim to teach everyone to build their own simple mobile application need. It is open source because many can share their knowledge and can share their code to improve it and so it will be working great from different point of view.
