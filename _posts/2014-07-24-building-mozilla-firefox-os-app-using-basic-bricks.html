In our Basic Bricks in the Mozilla Appmaker for Firefox OS Mobile Apps post we have been discussing about various basic bricks available for the Firefox OS application development. In this post we would like to make use of them so that we can experiment what we can do with the basic bricks. If you would like to know about the basic bricks check previous post.
In this post we would like to develop a simple Mozilla Love application. Which is developed with lot of love and passion. It will be great if you like to remix the application according to your need. We would like to go Step by step for developing the application.
First we would like to place Header component. After Placing the Header would like to change the Label as Love. Just click on the header and see the property will come on the right side, click there and edit.
After that I would like to place the Image component. The default height with 160px you can also change it, and dont forget give the source of the Image.
After that is the Text component. As of now I will wont be having any text and it will be blank. The alignment is made a center or left according to your wish. Increase or decrease the text size as you need and also the color.
Below the Text Component I would like to place the Counter. Which will be increasing the count whenever it gets the signal. The unit is declared as time by me. And the increment value is given as 1.
Then below them is a Button. Where I am giving Color of the text in White and the button color as red. Then the  Board cast value as "The above say's I Love Mozilla". So when I click the button it will be setting the text value as given above. 
Below button is the Metronome which will be giving the periodic signals. The output of this signals are feed to the counter. I have set as 60 signals per minute. And made it to Autostart. So when the page is loaded it will be starting to give the signals.
So when the page loads, the Metronome will be starting itself and the counter value will be keep on increase. When the Button is click a text will be appearing on the text box saying  "The above say's I Love Mozilla".
