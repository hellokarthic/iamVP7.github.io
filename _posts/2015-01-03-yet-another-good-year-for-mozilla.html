In my previous post Awesome Year of Mozilla Team I have mentioned the updates made by Mozilla during first half of the 2014. It was awesome to see lot of updates from them. Second half of 2014 is still more awesome when we compare to first.

Firefox OS Release in India

[caption id="attachment_356" align="alignright" width="238"] Firefox OS in India[/caption]

India is one of most awesome market. Here there are billion people with lot of dreams. It is really a huge consumer market compared to others. People love to have cheaper devices but should have good quality. Firefox OS has been launched in India with such a low cost, but it has gained lot of momentum, many people loved to buy it. The main mission of bringing next billion users online has been supported by Firefox OS.

Firefox Hello Preview

Video calling is one of the most awesome tool which we use in our date to day life. When we can include it with browser itself then need for third party apps will get reduced. Firefox hello preview has been released in this october, It is based on Real time Web RTC. We can easily talk with our friends with this.

Firefox 10th Birthday 

Firefox celebrated its 10th birthday this November. It was a big treat for the developers, for its birthday Firefox Developer edition came. Which has lot of developer tools and one of the important tool is Web IDE which is used for developers to developer Firefox OS applications easily.

Firefox OS in 30 Countries

Currently Firefox OS is reaching almost to 30 countries. We can expect few more to add in upcoming days. It is going to more awesome in the upcoming year.

For 2015 the year will be having pure focus on Firefox OS, Cloud Services and more security. It is going to be more awesome of Mozilla Followers. Lets wait for more updates in upcoming days.
