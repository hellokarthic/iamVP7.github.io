Many of us use Mobile phones for communicating with friends. For past few days our main usage of the phones are for Instant Messaging. In many developing countries like India Whatsapp has got very good name and many are using it. There are also other Instant Messaging Apps like Whatsapp and most common as Line, Telegram, Viber and so on.
Now in this post we are Going to see how to use Whatsapp for Firefox OS hand sets. There are many ways we can do. Here I would like to share the App which I have been using for past 2 months.
The application name is ConnectA2 which is available in many local languages. You can download from Firefox Marketplace
You can easily use this application as you have whatsapp in different platforms. When the screen is opened you can see the list of chats done by you. 
If you would like to start a new chat, start a new group or change your Profile setting and Config you can just click on the Down arrow button which is available at the right top. 
New Chat
When you click on the New chat it will be showing the list of people in your contact  you can easily select and one and start your chat.
New Group
On Click first it will be requesting the Name for the group which you would like to create. After Giving the name you can select the user who should be in that group and start chat.
Profile
Here you can easily change the profile Image you have and the Status which you you would like to set. And update the status.
Config
In this tab we can set our preference for Vibration and Notification in the background. We can also change the wallpaper when we need.
User details
When you chat with a person you would like to know about the user details. Just tap on the user name and see the chat you did. Again tap on his name at top so you will see his status and Mobile number details and configurations you set for him
Group details
Similar we can see who is the admin and who are all the participants in the Group. We need to follow same way we followed to see User details.
Whenever a photo/Video/Audio comes for the first time it will be requesting the permission to save in your Memory, if you accept only it will be saving.
