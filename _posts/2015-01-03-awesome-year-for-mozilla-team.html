Each and every year many organisation try to goto their next level, it is mandatory to go as people who use their service awaits for awesome change. Open source organisation are no exception, Mozilla has lot of products with them  which is widely used by many people for lot of reasons. This year also Mozilla team had lot of awesome updates and went to next level. For sure it is excellent year for the Mozilla team. Some of the updates would like to be shared. If you have more information please let me to know about it and help others to learn.

Firefox Accounts

Early Firefox had Firefox Sync simple version. It has been now enhanced to add some more special features called as Firefox Accounts with which we can register and share our bookmarks along different devices.

[caption id="attachment_251" align="aligncenter" width="297"] Firefox Accounts[/caption]

Cordova Apps for Firefox OS

Apache Cordova is one of the great library which is used by most of the developers to write cross platform HTML5 applications. Firefox OS teamed up with Cordova contributors and integrated Firefox OS with Cordova library so all developers can easily use them and build lot of apps for many platforms.

Australis Launch

One of the major products of Mozilla is their browser which is widely used by many people all around the world. We have seen a major redesign of Firefox Browser during April 2014. We had lot of new things in this update.

  • Elegant New Design
  • Customization Mode
  • Firefox Menu
  • Fun and Simple Bookmarks
  • Easy Access to Add-ons Manager
  • Enhanced Firefox Sync Powered by Firefox Accounts

[caption id="attachment_353" align="aligncenter" width="1316"] Australis browser's tab[/caption]

Emscripten for compiling to JavaScript

Firefox 31 has gave capabilities for developers to access Emscripten. It gave lot of opportunities for game developers and also first commercial web based game has been released this year.
