In Firefox OS app development point of view we have really a large number of tutorials. Which is available in the page

Very recently I have share a tutorial about getting started to Firefox OS app development. It is recommended to to check about it. The session is very small and you can learn very quickly.

First you have to create a small file in the folder where you have your website. Name the file as "manifest.webapp"

In that file you have to give the following lines

{
"name": "MozillaTN",
"description": "Mozilla Tamilnadu Website",
"launch_path": "/index.html",
"icons": {
"512": "/icons/icon512x512.png",
"256": "/icons/icon256x256.png",
"64": "/icons/icon64x64.png",
"128": "/icons/icon128x128.png"
},
"developer": {
"name": "Mozilla Tamilnadu",
"url": "http://mozillatn.github.io/"
}
}

detailed info of each key

  • "name" is the application name. It should be less than 12 characters.
  • "description" is where we have to tell about the application. It is good practice to have 25+ words and 50-
  • "launch_path" we have to mention which is going to be our first page when the app is launched.
  • "icons": this is the parent key which will have set of icons
    • "512": is we have to give the location of the icon of size 512x512 px.
    • "128": is we have to give the location of the icon of size 128X128 px.
  • "developer": will be having most of the information about the developer
    • "name" : we have to give developer name
    • "url": we can give the presonal website

Save the file in the same folder where all your website contents like index.html, css files and js files are there.

The next step is just open WebIDE

You  have to open the project. (Project -> Open packaged app)

You have to open the folder of your application. You can see it is valid. If it is invalid, then have to check any problem in manifest file.

Then you can zip all the contest and upload to the marketplace. Your app is done.
