Yesterday some of you  might have noticed the Google+ notification in desktop changed slightly. Previously we had only bell shaped icon if we have notifications then a red color will be added on the top of icon to show the number of unread notifications. Now the we have newly modified icon for +Google+  notifications in the +Gmail

When you clearly notice the icon we can see two different icons for the +Google+  Notifications.

Google+ icon on dark background
Google+ icon on light background

The above is the new icon(s) for Notification as said before. When you clearly notice the icon are differing very well. In the dark background we can see light color circle and dark bell while in the light colored background it is opposite, we can see dark color circle and light color bell.

+Google+ Developers  have done enough study and have written an best image processing algorithm so that it will detect the background image nature and depending on that image it will be displaying the icon. 
Purely this notification icon change had lot of background studies and it is awesome.
