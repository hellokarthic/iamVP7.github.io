Whenever it comes to any smartphones, it is little bit mandatory to know some of the gestures. In Firefox OS when you start your device for the First time you can simple the simple gestures available in the Firefox OS. It is very easy and can learn quickly.
For the first time it will ask whether it can take us to the Gesture tour or you can also skip it.
Starting page of Firefox OS gesture

To move between to different screens which has lot of interesting applications from Firefox Marketplace you simple have to tap you finger either from Left to Right or from Right to Left.
Moving Between Screens in Firefox OS

It is very easy to reposition any application in Firefox OS. Simple we have to hold the app Icon for a second. Then we can drag and position it as we need. To delete the app simple click on X mark.
Re positioning apps in Firefox OS

 One of the interesting feature in Firefox OS is adaptive search. We discussed about this before you can learn from my previous post.
Adaptive search in Firefox OS

Notification Screen in Firefox OS

 Closing an application in Firefox OS is done by holding the middle button of a second. It will be showing some of the running apps. We can just tap the screen from bottom to top.
Closing running application in Firefox OS

Thats it we have mastered some of the interesting gestures available in Firefox OS.
